## 🚀 Proposta do Projeto
O **Neumorphism.io** é um desafio técnico de reconstrução funcional guiado por Inteligência Artificial. A premissa central é assumir a posição de um **Desenvolvedor de Software assistido por IA** para reconstruir um aplicativo totalmente funcional baseando-se exclusivamente na observação da interface externa (UI/UX).

### O Diferencial Metodológico
Diferente do desenvolvimento tradicional, este projeto foi executado sob as seguintes restrições:
- **Black Box Reconstruction:** O código-fonte original do site de referência nunca foi visualizado.
- **Visual-to-Code:** A IA não recebeu o link direto; em vez disso, a lógica de negócio e a interface foram reconstruídas através de descrições estruturadas e engenharia de prompt baseada na observação do comportamento da UI.
- **Sincronia Homem-IA:** O desenvolvedor atua como o arquiteto que interpreta os estados visuais, enquanto a IA atua como o motor de implementação lógica e estilística.

## 🧠 Funcionalidades Reconstruídas
- **Gerador de Sombras Neumórficas:** Algoritmo para cálculo de luz e sombra (soft UI) baseado em cores de fundo.
- **Manipulação Dinâmica de Propriedades:** Ajuste em tempo real de curvatura, tamanho, distância e intensidade.
- **Output de Código:** Geração automática de sintaxe CSS para implementação imediata.

## 🛠️ Stack Tecnológica
- **Motor de IA:** Modelo Gemini (via Google AI Studio).
- **Interface:** HTML5 / CSS3 (Foco em variáveis CSS e Neumorfismo).
- **Lógica:** JavaScript para processamento de estados e reatividade.

## 📖 Fluxo de Desenvolvimento
1. **Análise de Interface:** Observação dos componentes, estados de hover, e comportamento dos seletores.
2. **Prompt Engineering:** Tradução das observações visuais em instruções lógicas para a IA.
3. **Refinamento Iterativo:** Ajuste fino dos parâmetros de iluminação e contraste para atingir a estética "Neumorphism" perfeita.
